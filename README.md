# Kawsar Portfolio Style Clone (Next.js)

This project recreates the content and very similar structure/style of the portfolio site at:

- https://www.kawsarlog.com/

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Notes

- Built with Next.js App Router.
- Content sections include hero, skills, services, projects, and contact.
- Styling is intentionally similar (dark theme + code-inspired UI), not an exact pixel-perfect copy.
